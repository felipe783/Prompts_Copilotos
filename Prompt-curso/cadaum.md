# Agent (Agente)
- Executa tarefas de forma mais “autônoma”
- Pode tomar decisões, rodar passos, usar ferramentas
- Ex: criar projeto, organizar arquivos, automatizar fluxo

👉 Use quando quer que a IA faça por você

---

# Plan (Planejar)
- Foca em estruturar antes de executar
- Cria passo a passo, roadmap, estratégia

👉 Use quando ainda está pensando como fazer

---

# Study (Estudar)
- Explica conteúdo de forma didática
- Pode quebrar em partes, dar exemplos, ensinar

👉 Use quando quer aprender algo

---

# Ask (Perguntar)
- Resposta direta e rápida
- Sem muita elaboração ou execução

👉 Use quando só quer resposta objetiva
